## project
- 项目名称: **慢递Mandy**
> 描述: 慢递（跨越时间的情感传递）：将你的思念、期待或秘密交付给未来。选择一个未来的日期，我们会在那时为你传递这份特别的情感。 基于：Sui + Seal + Walrus，确保未来日期来临时，只有你期待的人才能看到你传递的这份情感。
