## project
- 项目名称:SealVault
> 描述: SealVault 是一个构建在 Sui 区块链上的去中心化文件访问控制系统，结合了 Walrus 去中心化文件存储协议与 Seal 加密密钥管理框架。SealVault 的核心理念是：文件访问权 = NFT 持有权。用户上传文件，系统自动生成访问 NFT，只有拥有该 NFT 的账户才能访问对应文件，实现真正的 Web3 风格"资产即权限"模型。
